Step by Step Setup

Step 1 — Create GitHub Account

Go to github.com. Sign up for free. Create a new repository. Name it your website name. Example: myweddingphotography. Make it public. Check "Add README file". Click Create Repository.

Step 2 — Connect Netlify to GitHub

Go to netlify.com. Sign up with your GitHub account. Click "Add new site". Click "Import an existing project". Choose GitHub. Select your repository. Click Deploy Site. Netlify gives you a free URL like: myweddingphotography.netlify.app

Step 3 — Connect Your Custom Domain

Go to GoDaddy. Search for your domain name. Purchase it — around $12 to $13. Go back to Netlify. Site Settings → Domain Management. Add custom domain. Enter your GoDaddy domain. Netlify gives you DNS nameservers. Go back to GoDaddy. Manage DNS → Change nameservers. Enter the Netlify nameservers. Wait 24 to 48 hours. Your custom domain is live.

Step 4 — Install Windsurf

Go to windsurf.com. Download and install. Open your project folder. This is where your website files will live on your computer.

Step 5 — Tell Claude What to Build

Open claude.ai. Describe your website: "Build me a wedding photography website with a homepage, portfolio page, pricing page, about page, and contact form. Bright airy style. My name is [name]. Based in [city]." Claude writes a detailed Windsurf prompt. Copy it. Paste it into Windsurf. Windsurf builds your website.

Step 6 — Push to GitHub

In Windsurf type: git add -A, git commit -m "Initial website build", git push. Your website is now live.
